Module Countdown.Dictionary

module Dictionary: sig .. end
Dictionaries that support searching for longest words.

Type definitions


type t
The dictionary type, which encompasses a list of words.

Initialisation


val from_channel : Pervasives.in_channel -> t
Load a dictionary from an input channel.
val from_file : string -> t
Load a dictionary from a file.

Inspection


val print : t -> Pervasives.out_channel -> unit
Print all words in a dictionary to an output channel.

Filtering


val filter : t -> char list -> t
Only retain words that can be spelt using the given list of characters (multiple uses of a character are forbidden).

Searching


val longest : t -> string list
Return the longest word(s) in the dictionary.
val find_longest : t -> char list -> string list
Return the longest word(s) in the dictionary that are spelt using only the given list of characters (multiple uses are forbidden).

String conversions


val explode : string -> char list
Convert a string to a list of characters.
val implode : char list -> string
Convert a list of characters to a string.
