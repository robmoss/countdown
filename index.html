Countdown

Overview

This library implements solvers for both the Countdown.Letters and Countdown.Numbers rounds of the classic BBC show Countdown. It also provides a command-line interface (Play) for playing the games.


Countdown
Logic and solvers for playing the different Countdown rounds.
CLI
Command-line interface for playing the different Countdown rounds.
Play
Play Countdown rounds interactively.
Test
Test cases for the Countdown rounds.
